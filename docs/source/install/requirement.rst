.. _requirement: 

=======================================
動作環境
=======================================

OS

* Windows 10・Windows 11 (64 bit)
* AlmaLinux 8
* macOS Ventura(13)以降 (64 bit)（Intel/Apple M1対応）

  - 同梱している各計算エンジンの実行ファイルはmacOS上でのMPI並列実行に対応しておりません。

Java

* Java Runtime Environment 17

  - インストーラーに同梱されており、同時にインストールされます。
