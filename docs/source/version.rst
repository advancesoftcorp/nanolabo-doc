 - 機能追加

   - [Pro] QE: GIPAW法によるNMRスペクトル計算機能
   - QE: Car-Parrinello法分子動力学(CPMD)計算機能
   - MDL Molfile(.mol)読み込み対応

 - 変更・修正

 - その他、動作改善・バグ修正等

   - QE: フォノン計算の収束閾値設定
   - LAMMPS: 原子速度設定画面

   - リモートジョブ実行時のSSHセッション数を抑制

   - lsf2pbs/qsub変更（bsub呼び出し方法）

 - その他、動作改善・バグ修正等
